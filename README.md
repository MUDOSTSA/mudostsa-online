## System Overview

**MU-DOST SA Online** is a comprehensive management system for the Mapúa University Department of Science and Technology Scholars' Association (MU-DOST SA). This desktop application streamlines organization operations including event management, attendance tracking, inventory management, and member roster maintenance.

### Technology Stack

- **Frontend Framework**: Svelte (TypeScript)
- **Desktop Framework**: Tauri
- **Supported Platforms**: Linux, Windows

### Purpose

This system serves as a centralized hub for the organization's administrative needs, enabling officers and members to efficiently coordinate activities, maintain accurate records, and ensure smooth operations of the association.

## Platform Support

### Pre-built Releases
- ✅ **Windows** - Download the installer from the releases page
- ✅ **Linux** - Download the appropriate package from the releases page

### macOS
- ⚠️ **macOS (Apple)** - While Tauri supports macOS, pre-built binaries are not provided as the developers do not have access to Apple devices. macOS users can request the source code and build the application themselves using:
  ```bash
  npm run tauri build
  ```

## Repository Purpose

This repository contains official releases for the MU-DOST SA Online system. Each release includes packaged builds and deployment-ready files for Windows and Linux platforms.
